# Home

Cryptex is a virtual machine for executing custom byte code. 
It has the feature of binding functions from an external C# application.

All of this is done without reflection in order to guarantee Ahead-of-time(AOT) compilation support.

The byte code is not a language depended, thus ensuring various programming languages being supported.
